# ChildSpring 🌸

ChildSpring is an organization dedicated to building a child-friendly, parenting-literate, and neuro-inclusive society. We believe that with the right guidance, every child belongs, blooms, and discovers their unique strengths.

## 🚀 Live Demo
[https://childspringbd.com](https://childspringbd.com)

## 🛠️ Technology Stack
- **Framework:** [React 19](https://react.dev/) + [Vite](https://vitejs.dev/)
- **Styling:** [Tailwind CSS v4](https://tailwindcss.com/)
- **Routing:** [React Router DOM](https://reactrouter.com/)
- **Animations:** [AOS (Animate On Scroll)](https://michalsnik.github.io/aos/) + CSS Keyframes
- **Components:** [Lucide React](https://lucide.dev/) (Icons), [Swiper](https://swiperjs.com/) (Carousels)

## 📦 Setup & Installation

1. **Clone the repository:**
   ```bash
   git clone https://github.com/rhsohan/childspring.git
   cd childspring
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Start the development server:**
   ```bash
   npm run dev
   ```

4. **Build for production:**
   ```bash
   npm run build
   ```

## 🏗️ Deployment
This project is configured to automatically deploy to GitHub Pages on every push to the `main` branch via GitHub Actions.

The production deployment uses the custom domain `childspringbd.com`.
